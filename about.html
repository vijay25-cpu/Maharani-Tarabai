About This Project

This website is dedicated to Maharani Tarabai Bhosale, one of the bravest and most inspiring queens in Indian history. The purpose of this project is to highlight her leadership, courage, and contribution to protecting the Maratha Empire.

This project is created as an academic and educational website combining history with modern web design and interactive learning.

About the Creator

This website is designed and developed by a students as part of a college project. It demonstrates skills in HTML, CSS, JavaScript along with creativity, historical research, and user interaction.
